Como usar Visual Studio Code



Visual Studio Code es una aplicación que puedes usar para crear páginas web, en donde puedes poner lo que quieras y hacer aplicaciones para identificar sonidos, identificar imagenes, crear un juego en 2D, etc.También puedes ponerle funciones "boostraps" a esto se refiere a que dependiendo el dispositivo va cambiar el tamaña y si es una computadora o celular va a cambiar si es mas grande mas chico. Esta aplicación tambien tiene un cosola en la cual te puedes ayudar a saber si te equivocaste en algo.



Como usar la consola
